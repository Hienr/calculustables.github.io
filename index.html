General Formulas

C is defined as a constant

$$\large\frac{d}{dx}\ C\ =\ 0$$ 		$$\large\frac{d}{dx}\ cf(x)\ =\ cf'(x)$$
$$\large\frac{d}{dx}\ f(x)\ +\ g(x)\ =\ f'(x)\ +\ g'(x)$$ $$\large\frac{d}{dx}\ f(x)\ -\ g(x)\ =\ f'(x)\ -\ g'(x)$$

Product Rule

$$\large\frac{d}{dx}\ f(x)g(x) = f(x)g'(x) + g(x)f'(x)$$

Quotient Rule

$$\large\frac{d}{dx}\ \frac{f(x)}{g(x)} = \frac{g(x)f'(x) - f(x)g'(x)}{g(x)^2}$$

Chain Rule

$$\large\frac{d}{dx}\ f(g(x)) = f'(g(x))g'(x)$$

Power Rule

$$\large\frac{d}{dx}\ x^n = nx^{n-1}$$

Exponential and Logarithmic Functions

$$\large\frac{d}{dx}\ e^x = e^x$$ $$\large\frac{d}{dx}\ ln\ |x| = \frac{1}{x}$$
$$\large\frac{d}{dx}\ b^x = b^x\ ln\ b$$ $$\large\frac{d}{dx}\ \log_b x = \frac{1}{x\ ln\ b}$$

Trigonometric Functions

$$\large\frac{d}{dx}\ \sin(x) = \cos(x)$$ $$\large\frac{d}{dx}\ \csc(x) = -\csc(x)\cot(x)$$
$$\large\frac{d}{dx}\ \cos(x) = -\sin(x)$$ $$\large\frac{d}{dx}\ \sec(x) = \sec(x)\tan(x)$$
$$\large\frac{d}{dx}\ \tan(x) = \sec^2(x)$$ $$\large\frac{d}{dx}\ \cot(x) = -\csc^2(c)$$

Inverse Trigonometric Functions

$$\large\frac{d}{dx}\ \sin^{-1}(x) = \frac{1}{\sqrt{1-x^2}}$$ $$\large\frac{d}{dx}\ \csc^{-1}(x) = -\frac{1}{x\sqrt{1-x^2}}$$
$$\large\frac{d}{dx}\ \cos^{-1}(x) = -\frac{1}{\sqrt{1-x^2}}$$ $$\large\frac{d}{dx}\ \sec^{-1}(x) = \frac{1}{x\sqrt{x^2-1}}$$
$$\large\frac{d}{dx}\ \tan^{-1}(x) = \frac{1}{1+x^2}$$ $$\large\frac{d}{dx}\ \cot^{-1}(x) = -\frac{1}{1+x^2}$$
